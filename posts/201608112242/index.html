使用 jQuery 实现 AJAX 异步请求

2016-08-11 Web 前端

引用 jQuery 库文件。

<script type="text/javascript" src="js/jquery-3.1.0.min.js"></script>

在脚本中添加 ajax（GET方式）：

$.ajax({
2url:  'target.php',  // 请求路径
2type: 'get',  // 请求类型
2data: { 
22// 参数
22username: "caiyiming",
22password: "123456"
2},
2async : false,  // 是否异步执行
2success: function(response){
22// 成功后调用的代码
2},
2error: function(result) {  
22// 失败后调用的代码
2}  
});

使用 POST 方式：

$.ajax({
2url:  'target.php', 
2type: 'post',
2data: { 
22username: "caiyiming"
2},
2async : false,

2},

2}
