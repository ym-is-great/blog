当我们不希望一些服务端的文件被用户直接浏览、又必须用到这些文件时，可以通过 Apache Rewrite 禁止直接访问某些格式的文件。

首先，在根目录下的 .htaccess 文件（没有则新建）中开启重写引擎。

<!-- more -->

```
RewriteEngine on
```

然后，添加以下规则。这里禁止了用户访问 XML 格式的文件。

```
<FilesMatch "\.(xml)$">
  order deny,allow  
  deny from all
</FilesMatch>
```

实测 XML 被禁止访问后，仍然可以正常用 PHP 解析文件内部的数据。
