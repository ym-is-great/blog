在使用 textarea 控件时，某些场景下需要禁止用户使用回车换行。这时可以用 JavaScript 监听用户输入事件，判断按键为回车后返回 false 即可。

<!-- more -->

HTML:

``` xhtml
<textarea onkeydown="noWrap(event)"></textarea>
```

JavaScript:

``` js
function noWrap(event) {
	if (event.keyCode==13) event.returnValue = false;
}
```
