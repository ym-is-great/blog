网站发送验证码后，通常需要有一个倒计时器。在倒计时期间关闭“获取验证码”按钮，以防用户频繁地获取验证码。

我是利用 JS 的 setTimeout 方法实现倒计时的，基本示例如下，加上更新前端的代码即可完成倒数。

<!-- more -->

``` js
var time = 60;
document.write(time--+"<br>"); 
setCountdown(time);

function setCountdown(time){
	document.write(time--+"<br>"); 
	setTimeout(function(){ setCountdown(time) },1000);
}
```
