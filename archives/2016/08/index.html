检测浏览器是否支持 H5

通过全局对象检测：

<script>
  window.onload = function() {
    if (window.applicationCache) {
      document.write("Yes, your browser can use offline web applications.");
    } else {
      document.write("No, your browser cannot use offline web applications.");
    }
  }
</script>
