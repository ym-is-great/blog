经过实测自适应效果非常好，但该方案仍然存在 BUG ，在手机端的一些浏览器上会在页面下方出现白边。

经测试发现这是因为背景图铺满的区域是 html 标签，而不是我们实际感知到的屏幕范围。有的浏览器会将 html 标签里的内容的高度作为 html 标签的高度，如果 html 的高度小于设备屏幕的高度，就会出现空白区域。解决方法也很简单，在 CSS 代码中设置 html 的高度始终为100%。
